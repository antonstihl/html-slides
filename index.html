HATEPRES

Hypermedia As The Engine of Presentation

A presentation library

HATEPRES provides tooling to render semantic HTML as a delightful visual aid for your presentation.

  1. Wrap all sections (slides) in a div with id slide-deck 
                
<div id="slide-deck">
  <section>...<section>
  <section>...<section>
</div>
  2. Add the head style tag
  3. Add the head script tag, deferred
  4. (Optional) add a cpad element, <pre id="cpad"></pre>
  5. Add attribute hp-f (focusable) or hp-fr (focusable revealable) to make elements focusable and focusable+revealable, respectively.

Philosophy

Some of the key characteristics of HATEPRES come from David JP Phillips TEDx talk, How to avoid death by PowerPoint.

    • One message per slide
    • Use contrast and size to direct attention
    • Don't write sentences
    • Dark bakground
    • Maximum of 6 items per slide
